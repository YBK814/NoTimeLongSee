# NoTimeLongSee

두플러스의 주간 뉴스레터 **「노타임롱씨(No Time, Long See)」**의  
레이아웃 구조와 UI 정리를 위한 레포지토리입니다.

본 레포지토리는 **이전 버전 뉴스레터 레이아웃**과  
**현재 운영 중인 뉴스레터 레이아웃**이 함께 공존하는 공간으로,  
뉴스레터의 일관성과 확장성을 유지하기 위한 기준점 역할을 합니다.

---

## 📰 About NoTimeLongSee

- **발행 주기**: 주 1회
- **구독자 수**: 약 **13,000명**
- **운영 주체**: 두플러스
- **목적**:  
  바쁜 일상 속에서도 놓치지 말아야 할 인사이트를  
  짧고 명확한 구조로 전달하는 주간 뉴스레터

---

## 🎯 Repository Purpose

이 레포지토리는 다음을 목표로 합니다.

- 뉴스레터 **레이아웃 구조의 체계적 관리**
- 이전/현재 레이아웃의 **비교 및 히스토리 보존**
- 향후 개편을 고려한 **확장 가능한 레이아웃 설계**
- 디자이너·개발자 간 **공통 기준 문서화**

---

## 🗂️ Layout Structure

본 레포지토리에는 다음 두 가지 레이아웃이 포함됩니다.

### 1️⃣ Legacy Layout
- 이전에 사용되던 뉴스레터 레이아웃
- 당시 콘텐츠 흐름과 디자인 방향성 보존
- 레퍼런스 및 히스토리 관리 목적

### 2️⃣ Current Layout
- 현재 실제 발송 중인 뉴스레터 레이아웃
- 구독자 가독성과 콘텐츠 전달 효율을 고려한 구조
- 유지보수 및 개선 대상
